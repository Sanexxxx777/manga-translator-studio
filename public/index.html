        Чем открыть .cbz на Mac
        
    
          
  1. 
            YACReader — лучший OSS-ридер для CBZ на macOS. Закладки, листание разворотом, библиотека.
            brew install --cask yacreader
          
    2. 
          
  2. 
            Simple Comic — нативный, лёгкий, открывается за секунду. App Store, бесплатно.
          
    3. 
          
  3. 
            Chunky Reader — есть Mac-версия, удобный жестовый интерфейс. Платный, ~$5.
          
    4. 
        

        
CBZ — это просто zip с картинками. В крайнем случае можно переименовать в .zip и открыть как обычную папку.
